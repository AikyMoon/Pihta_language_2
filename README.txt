В языке Пихта(Pihta) пока что есть несколько функций(в дальнейшем будут добавляться новые)

1. var имя_переменной = значение_переменной;(при создании новой переменной или
обновления значения нужно указать var(переменная), затем имя, а затем значение после =)
2. console имя_переменной/значение;
(console может пока что выводить только одно значение или одну переменную)

Язык Пихта пока что умеет работать только с числовыми значениями
И тоже важная пометка действия выполняются слева направа(нет учета порядка действий), например
1 + 2 * 3, то значение будет 9
Также при создании переменной со знаками знаки и значения или имена перменной нужно писать через пробелов
var i = 1;
var j = 2;
var k = i + j; правильно
var k = i+ j; не правильно

И в конце строки надо указывать ;
В код была добавлена обработка одной пары круглых скобочек т.е.
var k = (i + j) / 10; сработает, если будут вложенные скобочки, тогда не сработает.
И писать скобочки нужно как в примере выше, т.е. после открывающей скобочки пробелов нет и перед закрывающей тоже

Также в структуру языка добавился ввод(команда entry вводится без каких-либо скобочек и структура trigger)
как писать структуру trigger есть в файле test.fir

После скачивания архива и его открытия вы запускаете файл main.py
Программа будет ожидать файл с расширением .fir
После введение названия файла нажмите Enter и код выполниться, чтобы закрыть окно выполнения,
после того как код выполнился также нажмите Enter
Файл с примером будет лежать в папке Examples/
Файлы с кодо редактируются через блокнот или другой текствый редактор

Чтобы запустить main.py на машину, на которой будет выполняться файл
должн быть установлен python 3.7 и выше
